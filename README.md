This is code repository of [https://wycode.cn](https://wycode.cn "https://wycode.cn")

## build

this project depends on 
- `nodejs` (8.0 at least)
- `hexo`

```bash
npm install hexo-cli -g
hexo clean
hexo g
```

## serve on localhost

```bash
hexo s
```
